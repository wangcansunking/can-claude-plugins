built-in-comment-ui

Add a text-selection-based comment UI to the plan-harness HTML dashboard. Reviewers highlight text in any rendered plan doc and leave comments Word-style, stored as JSONL on the host filesystem and attributed to the logged-in reviewer. The author (host) gets a second action — Request update — that hands the selection back to a Claude Code agent as a revise directive.
